Navigator.getUserMedia

  • Camera
  • Microphone
  • Screen sharing
  • User permission required
  • Javascript API
  • Required user permission so no funny business

Browser Support
Desktop
Mobile
  • Good old Internet Explorer

Example


var getUserMedia = navigator.getUserMedia ||
                   navigator.webkitGetUserMedia ||
                   navigator.mozGetUserMedia;
getUserMedia(
    {
        audio: true,
        video: true  // can also be a config object
    },
    function(stream) {
        var video = document.querySelector('video');
        video.src = window.URL.createObjectURL(stream);
        video.onloadedmetadata = function(e) {
            video.play();
        };
    },
    function(err) {
        console.log("The following error occurred: " + err.name);
    }
);
  • Vendor prefixes required
  • When W3C specification is introduced the browser may implement a version before it reaches Candidate Recommendation status
  • Config object, success handler, error handler
  • Error if user denies access

Example

  • Asks for my permission
  • Once granted it streams to the video element

Device Orientation

Accelerometer, Gyroscope & Magnometer

  • Movement
  • Orientation
  • Compass

Orientation

alpha
the number of degrees
the device is being twisted
around the center of
the screen gamma
the number of degrees
the device is turned
left or right beta
the number of degrees
the device is tipped
forward or backward
  • Alpha - spinning
  • Beta - flipping over head to front
  • Gamma - rolling over on side

Example


window.addEventListener("deviceorientation", function(event) {

    var alpha = event.alpha; // degress twisted around center
    var beta = event.beta; // degrees tipped forward or backward
    var gamma = event.gamma; // degrees turned left or right

    // do something awesome here

}, true);
  • Subscribe to the deviceorientation event
  • Event object contains all information

Browser Support
Desktop
Mobile
  • Depends on having gyroscope, accelerometer on device
  • My MacBook doesn't
  • Load demo!

Geolocation

  • Access user latitude and longitude
  • Permission required

Browser Support
Desktop 9+
Mobile
  • Not bad!
  • Well done Internet Explorer

Example


navigator.geolocation.getCurrentPosition(success, error);

function success(position) {

    var latitude  = position.coords.latitude;
    var longitude = position.coords.longitude;

    // do something clever here

}

function error() {
    console.log('Unable to get location');
}
  • Pass it success, error callback
  • Error callback if something went wrong or user denied access

Uses

  • Find nearest [something]
  • Plot user location on map - real time tracking
  • Directions - to/from current location
  • Geocaching
  • Practical like mapping
  • Bit more fun like geocaching

Light Sensor

  • Gives light intensity value in lux
    • 10 ~ 50 lux: Dim Environment
    • 100 ~ 1000 lux: Normal
    • > 10000 lux: Bright

Uses

  • Change page theme depending on light level
  • In bed reading in the dark you don't want to be blinded by a bright white background
  • Google Maps uses this - detects light levels and shows you either a dark map with blue roads or light map with white roads

Browser Support
Desktop
Mobile
  • Only Firefox

Example


window.addEventListener('devicelight', function (event) {

    var luminosity = event.value;

    // do something blindingly good here

});

Proximity

  • Detect when a user is close to the screen
  • Firefox only :(

window.addEventListener('userproximity', function(event) {
    if (event.near) {
        console.log('user is near');
    }
    else {
        console.log('user is not near');
    }
}, true);
  • You've all used a proximity sensor
  • When making phone call - put phone to head, screen turns off
  • Take phone away from head, screen turns back on

Vibration

Chrome, Firefox and Opera


// Vibrate once for one second
navigator.vibrate(1000);

// Vibrate for 3 seconds, wait 2 seconds, then vibrate for 1 second
navigator.vibrate([3000, 2000, 1000]);
  • Chrome, Firefox and Opera support

Battery Status

Chrome, Firefox and Opera


navigator.getBattery().then(function(battery) {
    battery.charging; // true/false
    battery.level; // % displayed as decimal from 0 to 1
    battery.chargingTime; // time spent charging
    battery.dischargingTime; // time spent not charging
});
  • Chrome, Firefox and Opera support
